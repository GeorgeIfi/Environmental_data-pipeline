# Data Dictionary

## Fields
- **date**: Date of air quality measurement (YYYY-MM-DD)
- **location**: Monitoring station location (Newcastle Centre)
- **latitude/longitude**: Geographic coordinates of monitoring station
- **site_type**: Type of monitoring site (Urban Background)
- **zone**: Geographic zone (North East)
- **pollutant_type**: Type of pollutant measured (black carbon, particulate matter at various wavelengths)
- **value**: Pollutant concentration measurement
- **unit**: Measurement unit (ugm-3 = micrograms per cubic meter)
- **status**: Data verification status (N = Not Verified)
